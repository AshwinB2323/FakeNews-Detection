# Fake News Detection using Deep Learning

## Overview
Fake news detection is an important task in combating misinformation online. This project applies **deep learning techniques** to classify news articles and social media content as **real or fake**. Using multiple datasets, we preprocess text data and train models to distinguish between legitimate and misleading information.

This project is part of a **Deep Learning Course**, fulfilling the requirement of **training a model on unstructured data with an original component**—namely, comparing different datasets and applying transformer-based architectures.

---

## Datasets
I use three datasets for fake news classification:

1. **CoAID** (COVID-19 Healthcare Misinformation Dataset)  
   - Includes news articles and tweets related to COVID-19 misinformation.  
   - Contains real and fake labels.  
   - [Source](https://github.com/cuilimeng/CoAID)

2. **FakeNews-Kaggle Dataset**  
   - General fake news dataset collected from various online sources.  
   - Provides both article text and headlines.  
   - [Source](https://www.kaggle.com/c/fake-news)

3. **LIAR Dataset**  
   - Politifact-based dataset containing short statements labeled as true or false.  
   - Includes speaker metadata and fact-checking details.  
   - [Source](https://www.cs.ucsb.edu/~william/data/liar_dataset.zip)
